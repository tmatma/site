Intro to Jade

Jade is a language which compiles to HTML.

This will not be an exhaustive document covering Jade but mainly covering conventions you'll see used in Kickstrap. For complete documentation, visit the Jade homepage.

Like CoffeeScript and Stylus, this language substitutes indentation for noisy characters.

For example, instead of 

<div>
  <span>
    <nav>
    </nav>
  </span>
</div>

We can just write

div
  span
    nav

The closing of a tag is implied by beginning a new line indented back in. So if we wanted to say "hello" in the span, but not the nav, we'd say

div
  span
    | hello
    nav

The pipe (|) is a special character we use to tell jade that "hello" is not an html tag. Otherwise if we did this:

div
  span
    hello
    nav

We'd have this:

<div>
  <span>
    <nav>
    </nav>
    <hello>
    </hello>
  </span>
</div>

Writing inside tags

Commonly, you'll need to add classes and ids to your tags. This is done using CSS syntax where dots (.) are classes and hashmarks (#) are ids.

div.menu-item#main-menu

Would yield

<div class="menu-item" id="main-menu"></div>

Other attributes can be encapsulated by parentheses as such:

div.menu-item#main-menu(ng-click="activate()")

Yields:

<div class="menu-item" id="main-menu" ng-click="activate()"></div>

Compiling in Roots

Kickstrap uses Roots to make the process of compiling Jade, CoffeeScript, and Stylus seamless as well as organizing partials and resources.

To compile your site while working on it, run kickstrap watch. This is exactly the same as running roots watchand is duplicated for ease-of-use with other kickstrap commands.

For production, run kickstrap compile. Again, this is the same as running roots compile.

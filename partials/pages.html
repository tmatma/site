Creating Pages

To create a page, make a new .jade file in views/. Better, use the provided newpage.jade, also in views/.

Pages that use the global layout.jade should have this at the top

extends layout
 
block content

Obviously, the first line tells Jade this is extending layout.jade.

If you open layout.jade, you'll see there is a block content defined there as well.

extends layout
 
block content
  h1 hello world

This above code is telling Jade it should put the indented code there when it extends layout. You can create other blocks in layout as well.

For partials, omit both the extends and block line completely. This will default to extending blank.jade. Because partials are used in pages already extended by layout, there's no need to write in the extra resources.
